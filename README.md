# WordPress Project

Этот репозиторий содержит код темы WordPress, конфигурацию Docker и инструкции по работе с проектом.

## 🚀 Как развернуть проект локально

### 1️⃣ Клонирование репозитория
```sh
git clone https://github.com/ТВОЙ_НИК/РЕПОЗИТОРИЙ.git
cd РЕПОЗИТОРИЙ
```

### 2️⃣ Запуск Docker-контейнеров
```sh
docker-compose up -d
```
Проект будет доступен по адресу: **http://localhost:8080**  
phpMyAdmin: **http://localhost:8081**

### 3️⃣ Вход в админку WordPress
Перейти в **http://localhost:8080/wp-admin** и войти в систему.

## 🔧 Как вносить изменения в код

### 1️⃣ Редактирование темы
Вся кастомизация находится в папке:
```
wp-content/themes/ТВОЯ_ТЕМА/
```
После изменений просто обнови страницу в браузере
